    React.isValidElement

    React 提供了isValidElement() 方法，用于判断传入对象是否是有效的 ReactElement。

    使用方法

    const div = React.createElement('div');
    React.isValidElement(div);
    // true
    React.isValidElement(document.getElementById('example'));
    // false

    比较多用在 ES5 的语法环境中。
